CLI Trivia Game 

A simple and interactive Command Line Interface (CLI) trivia game built with Node.js. Answer multiple-choice questions within a time limit and test your knowledge.

 Features
Multiple-choice trivia questions
15-second countdown timer per question
Instant feedback on answers
Final score with performance message
Runs entirely in the terminal
Tech Stack
JavaScript (Node.js)
Built-in readline module
 Getting Started
Prerequisites
Install Node.js
Run the Game
node trivia-game.js
 How It Works
Each question displays 4 options
Enter the number (1–4) for your answer
You have 15 seconds per question
If time runs out, the game moves on automatically
Example
Question 1: What is the capital of France?
1. Berlin
2. Madrid
3. Paris
4. Rome

Your answer: 3
Correct!
Scoring
Each correct answer earns 1 point
Final score is shown at the end
